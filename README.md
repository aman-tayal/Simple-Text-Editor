# 📝 Simple Text Editor

A beginner-friendly desktop application developed using **Python** and **Tkinter** that demonstrates the fundamentals of GUI development and file handling. The application enables users to create, edit, open, and save text files through a menu-driven interface, providing a practical introduction to event-driven programming and desktop application development.

## 🚀 Features

* Create new text files
* Open existing text files
* Save text files
* Edit text content
* Menu-driven interface
* Exit the application

## 🛠️ Technologies Used

* Python
* Tkinter

## ▶️ Getting Started

1. Clone the repository.
2. Navigate to the project directory.
3. Ensure Python 3 is installed.
4. Run the application:

```bash
python text_editor.py
```
